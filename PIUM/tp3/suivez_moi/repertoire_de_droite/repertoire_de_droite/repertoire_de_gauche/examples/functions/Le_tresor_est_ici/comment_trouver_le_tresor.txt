


Vous vous trouvez dans un répertoire en désordre complet. Essayez de
déterminer les différents types de fichiers qui se trouvent dans le
répertoire. Créez pour chaque type de fichier un répertoire et copiez les
fichiers de ce type dans le répertoire.

Il parait qu'il y aurait pusieurs fichiers qui contiennent du code source.
Pour ces fichiers créez un répertoire 'src' et comme sous-répertoires
utilisez les suffixes des différents fichiers de code source (c, cpp, py).

Au boulot !
